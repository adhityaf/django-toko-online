
# Django Toko Online

An ecommerce app build with Django and Bootstrap 5 


## Features

- Admin Panel
    - Manage accounts
    - CRUD category
    - CRUD product
    - CRUD variations
    - Manage cart and items
- Authentication
    - Register
    - Login
    - Logout
    - User activation using email
    - Forgot password
- User Pages
    - Index page
    - Product page
    - Search
    - Product details
    - Manage Cart (Add to cart, increase quantity, decrease quantity, remove an item)
    - Filter by category
